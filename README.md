# Superstore Sales EDA — Python

## Tools: Python | Pandas | Matplotlib | Seaborn | Jupyter Notebook

## Objective
Perform exploratory data analysis on Superstore sales data to uncover 
revenue drivers, loss areas, and discount impact on profitability.

## Key Findings
- High discounts (>40%) consistently generate negative profit
- Technology leads in both sales and profit
- Furniture category needs pricing/cost review
- West region is the strongest market
